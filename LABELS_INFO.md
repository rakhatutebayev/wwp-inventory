# Печать наклеек с QR-кодами

## Поддерживаемые форматы наклеек

Система поддерживает следующие популярные форматы наклеек для печати инвентарных номеров с QR-кодами:

### 1. 38x21 мм (Avery L7159)
- **Размер:** 38 мм × 21 мм
- **Количество на листе A4:** 24 наклейки
- **Расположение:** 4 колонки × 6 строк
- **Применение:** Маленькие устройства, компактные наклейки

### 2. 50x25 мм
- **Размер:** 50 мм × 25 мм
- **Количество на листе A4:** 21 наклейка
- **Расположение:** 3 колонки × 7 строк
- **Применение:** Стандартный размер для большинства устройств

### 3. 70x36 мм (Avery L7160)
- **Размер:** 70 мм × 36 мм
- **Количество на листе A4:** 12 наклеек
- **Расположение:** 3 колонки × 4 строки
- **Применение:** Крупные устройства, более читаемый текст

### 4. 100x50 мм
- **Размер:** 100 мм × 50 мм
- **Количество на листе A4:** 8 наклеек
- **Расположение:** 2 колонки × 4 строки
- **Применение:** Очень крупные устройства, максимальная читаемость

## Как использовать

### Печать одной наклейки
1. Откройте страницу устройства
2. Нажмите кнопку "Печать наклейки"
3. Выберите формат наклейки
4. Нажмите "Печать"
5. В открывшемся окне нажмите Ctrl+P (или Cmd+P на Mac) для печати

### Печать нескольких наклеек
1. Откройте список устройств
2. Выберите устройства с помощью чекбоксов
3. Нажмите кнопку "Печать наклеек (N)"
4. Выберите формат наклейки
5. Нажмите "Печать"

## Что печатается на наклейке

- **QR-код:** Содержит инвентарный номер устройства
- **Текст:** "Инв. №: [номер]"

QR-код можно отсканировать мобильным приложением для быстрого поиска устройства.

## Рекомендации по печати

1. **Бумага:** Используйте специальную бумагу для наклеек соответствующего формата
2. **Принтер:** Рекомендуется использовать лазерный принтер для лучшего качества
3. **Настройки печати:** 
   - Масштаб: 100% (без масштабирования)
   - Размер бумаги: A4
   - Поля: Минимальные или без полей
4. **Тестовая печать:** Сначала распечатайте одну наклейку для проверки размера

## Технические детали

- QR-код генерируется с уровнем коррекции ошибок M (Medium)
- Размер QR-кода автоматически подстраивается под размер наклейки
- QR-код содержит только инвентарный номер для быстрого сканирования
- Формат QR-кода соответствует стандарту ISO/IEC 18004

## API Endpoints

- `GET /api/labels/print?device_ids=1,2,3&format=38x21` - Генерация HTML страницы для печати
- `GET /api/labels/qr/{device_id}?size=200` - Получить QR-код устройства в формате base64
